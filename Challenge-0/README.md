# Challenge 0: Tokenization

This challenge is part of the Speedrun Ethereum curriculum. It demonstrates how to build an NFT minting and transferring dApp using an ERC-721 smart contract (YourCollectible).

## Overview

The goal of this project is to understand on-chain ownership, compile and deploy smart contracts, and interact with them via a Next.js frontend. 

The application allows users to mint unique NFTs, view their holdings, and transfer their NFTs to other addresses. It is built using **Scaffold-ETH 2**, with Hardhat for smart contract development.
